<H4>

  * scan.py: Pseudo-code for simple scan 1st version.
* scan_simple.py: Code for simple scan, using python turtle for visualization. It logs data to a file (please replace path).
* scan_simple_console_log.py: The same as scan_simple.py, but it logs data to console.

Libraries needed:

* numpy
* turtle
* astropy (https://docs.astropy.org/en/latest/install.html)
* jplephem (https://pypi.org/project/jplephem/) -> install with pip install jplephem
</H4>
